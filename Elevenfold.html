Semi-autonomous CargoMate for Factory and Warehouse Automation

Introduction

The Semi-autonomous CargoMate is an innovative project designed to automate material transportation in factory and warehouse settings. Leveraging TensorFlow's object detection API, the vehicle interprets visual cues such as directional signs, stop signals, and speed limits to navigate autonomously. This system reduces the need for human intervention, enhancing operational efficiency and safety while lowering labor costs. Its AI-driven vision system offers adaptability to dynamic environments, making it an ideal solution for modern smart factories.

This project addresses the growing demand for automation in industrial logistics by integrating computer vision and machine learning to create a flexible, cost-effective navigation system. The CargoMate prototype has successfully demonstrated autonomous navigation in controlled environments, showcasing its potential to transform material handling processes.

Project Goals

  • Develop a functional semi-autonomous vehicle for material transport.
  • Implement intelligent navigation using visual cues for directional decisions.
  • Integrate real-time traffic sign recognition (e.g., stop, turn, speed limits).
  • Ensure reliable obstacle detection and avoidance for safe operation.
  • Enhance operational efficiency and safety in industrial environments.
  • Facilitate adaptability to changing factory layouts using machine learning.

Key Features

Vision-Based Navigation

  • Uses a camera and TensorFlow Lite to detect navigation signs in real-time.
  • Processes visual data to make autonomous movement decisions.

Real-Time Sign Detection

  • Detects and interprets signs like 'Stop', 'Turn Left/Right', and 'Slow Down'.
  • Triggers corresponding vehicle actions with high accuracy.

Obstacle Avoidance

  • Utilizes an ultrasonic sensor for basic obstacle detection.
  • Ensures safe navigation by avoiding collisions with dynamic objects.

Adaptability

  • Learns from operational data to improve navigation over time.
  • Adapts to changing factory layouts without infrastructure modifications.

Technical Architecture

The CargoMate's architecture is designed for modularity and real-time performance:

  • Camera Module: Captures live video for sign detection.
  • Raspberry Pi 4B: Processes video, runs object detection, and sends commands.
  • Arduino Nano: Controls motors and interfaces with sensors.
  • Motor Driver (BTS7960): Manages motor speed and direction.
  • Ultrasonic Sensor: Provides obstacle detection data.
  • Software Stack: Includes Python, OpenCV, TensorFlow Lite, and Arduino IDE.

The system uses a continuous control loop: the camera feeds video to the Raspberry Pi, which detects signs and sends commands to the Arduino for motor control. This ensures real-time responsiveness to environmental changes.

Setup Instructions

Hardware Assembly

  • Mount Raspberry Pi, Arduino, motor driver, and sensors on the chassis.
  • Connect DC motors to the motor driver and wheels.
  • Attach the webcam and ultrasonic sensor to the Raspberry Pi and Arduino.
  • Power the system using a LiPo battery and boost converter.

Software Installation

  • Install Raspberry Pi OS and required Python libraries (OpenCV, TensorFlow Lite).
  • Program the Arduino using the Arduino IDE for motor and sensor control.
  • Set up the object detection model on the Raspberry Pi.
  • Integrate all components and test the control loop.
