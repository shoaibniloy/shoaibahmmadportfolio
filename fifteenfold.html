Vision-Based Bee Behavior – Real-Time Honeybee Sensing GUI

Overview:

A PyQt6 desktop application for real-time monitoring of hive activity using YOLOv11 (Ultralytics) for detection and ByteTrack for multi-object tracking. The tool detects bees, mites, pollen, queen, queen_cell, and varroa, then logs behavior, traffic counts, and anomalies. Optional ONNX behavior models and Isolation Forest can enrich insights.

Features:

  • Real-Time Detection & Tracking: YOLOv11 + ByteTrack with robust class aliasing.
  • Behavior Insights: entrance/exit counting, queen tracking, optional behavior classification via ONNX or heuristics.
  • Health Monitoring: dead-bee detection, pest (mite/varroa) statistics, anomaly detection, swarm-risk estimation.
  • Interactive GUI: live annotated video feed, metrics panel with counters and reset controls.

Getting Started:

  1. Clone: 
    git clone https://github.com/shoaibniloy/Vision-Based-Bee-Behavior-.git
cd Vision-Based-Bee-Behavior-
  2. Install (Python 3.10+ recommended): 
    pip install -r requirements.txt
    Includes: PyQt6, opencv-python, ultralytics, supervision, scikit-learn, onnxruntime (optional).
  3. Add your trained YOLOv11 weights as bee.pt in the project root (classes: bee, mite, pollen, queen, queen_cell, varroa).
  4. Run: 
    python bee-vision.py
    Defaults to webcam 0; switch to a video via the GUI’s “Open File…” option.
