Robotic Arm 4DoF Project

Overview:

This robotic arm, equipped with three open-loop stepper motors and controlled by an ESP32-based development board, offers precise control and adaptability for a range of tasks. Its design incorporates 3D-printed parts, a customizable grabber, and multiple operating modes, making it suitable for dynamic, on-the-fly applications in various industries.

Components and Construction:

  • Stepper Motors
  • 3D-Printed Parts
  • ESP32 Development Board
  • Customizable Grabber

Functionality:

    Supervised Mode
  • Supervised Mode: Controlled remotely via a web application hosted on the ESP32. The user can input angles, speeds, and movement sequences, reprogramming the arm for different tasks without coding.
    • Imitation Mode
  • Imitation Mode: Uses a camera to detect colored objects. The image data is processed by OpenCV, and the coordinates are sent to the Arduino to move the arm to grab the object automatically.
    • Imitation Mode
  • Precise Joystick Control: A DS4 PlayStation controller connects via Bluetooth for manual control, allowing for real-time adjustments to the arm's position.

Custom Controller:

  • Web Application Control: The ESP32-based system is accessed via a web app, allowing global control of the robotic arm. The app features easy-to-use input forms for angle, speed, and movement sequence settings.
  • Wireless Communication: Using Bluetooth, the robot can be controlled remotely, enabling flexibility and efficient control without the need for wired connections.

Achievements:

This project highlights cutting-edge technology in robotics, combining custom hardware and software to offer precise control, automation, and flexibility. Its versatile modes and remote capabilities make it a valuable tool for industries requiring dynamic task execution.
